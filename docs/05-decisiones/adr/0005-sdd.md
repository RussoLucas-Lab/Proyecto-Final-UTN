# ADR-0005: Spec Driven Development + iteración ágil

**Estado:** Aceptada · **Fecha:** 2026-06

## Contexto
El proyecto integra procesos heterogéneos y se desarrolla con asistencia de herramientas de IA (Claude Code, Cursor, Copilot), que rinden mejor con especificaciones precisas.

## Decisión
Adoptar **SDD**: especificar cada funcionalidad antes de implementarla; combinarlo con iteración ágil (ciclos cortos, revisión periódica). Las specs de este repositorio son la fuente de verdad.

## Consecuencias
- (+) Reduce ambigüedad y mejora el output de las herramientas de IA.
- (+) Trazabilidad requerimiento → spec → código.
- (−) Costo inicial de redacción de especificaciones; se compensa en mantenibilidad.
